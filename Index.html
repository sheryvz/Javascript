Javascript

Jvazcript es un lenguaje de programación dinamico y de alto nivel para crear interfaces interactivas y dinámicas

es uno de los tres pilaresfundamentales del diseño web

  • HTML -> Define la estructura del contenido. ( Elementos que aparecen en la página )
  • CSS -> Define la apariencia (colores, tamaños, estilos)
  • Javascript (JS) -> Define el comportamiento (lo quer hace la página o sus elementos.)

Caracteristicas de Javascript

  1. Lenguaje interpretado -> No necesitamos compilarlo. El navegador lo entiende directamente
  2. Basado en objetos -> Maneja objetos como document,window,arrays, date, class.
  3. Dinámico -> Podemos modificar el contenido en tiempo real. (Sin recargar)
  4. Versatil y de facil aprendizaje -> Aunque nacio para la web, se usa también en servidores, aplicaciones móviles. videojuegos y hoy en día Inteligencia Artificial.
